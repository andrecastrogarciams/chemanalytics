## Story

**As a** administrador do sistema,  
**I want** autenticação com perfis de acesso mínimos,  
**so that** apenas usuários autorizados possam operar cadastro, revisão e consulta.

## Acceptance Criteria

1. A API suporta login seguro e emissão de credencial de sessão ou token conforme padrão adotado.
2. Existem perfis mínimos `admin`, `reviewer` e `consulta` com regras de acesso coerentes.
3. O primeiro acesso ou reset de senha exige troca de senha quando aplicável.
4. Endpoints e telas protegidas respondem com erro adequado quando o perfil não possui autorização.
5. Há testes cobrindo autenticação bem-sucedida, falha de login e bloqueio por permissão.
