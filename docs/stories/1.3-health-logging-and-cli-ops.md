## Story

**As a** equipe de operação interna,  
**I want** health checks, logs e comandos operacionais básicos,  
**so that** eu consiga diagnosticar problemas e operar o sistema sem depender da interface web.

## Acceptance Criteria

1. Existe endpoint ou comando de health para aplicação e dependências críticas.
2. O sistema gera logs mínimos para autenticação, sincronização, execução de conferência e erros.
3. Há pelo menos um fluxo operacional validável via CLI para diagnóstico.
4. A estratégia de logs e health está documentada para uso local e servidor.
5. O comportamento de falha em dependências críticas é observável de forma objetiva.
