## Critérios de organização

- Epics agrupam entregas coesas de valor.
- Stories foram quebradas para caber em implementação incremental.
- Cada story inclui dependências, acceptance criteria, testing, checklist e file list.
- O backlog está ordenado para respeitar `CLI First -> Observability Second -> UI Third`.
